# Email-spam-classifier

Building an Intelligent SMS Spam

Filter!

Ever wondered how your phone knows which messages are junk? I just built a Spam Classifier using Python!

Using the SMS Spam Collection dataset, I implemented a pipeline to:

Clean and preprocess raw text data.

Vectorize messages using Bag-of-Words.

Train a Multinomial Naive Bayes model to predict 'Ham' vs. 'Spam'.

It's amazing to see how effectively machine learning can filter out noise!
